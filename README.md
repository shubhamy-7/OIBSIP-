# Web Dev L1 — Task 1: Landing Page

**Track:** Web Development & Designing — Level 1
**Task:** Landing Page
**Author:** Shubham Yadav

## Objective

A visually polished static landing page for a fictional small-batch coffee roastery, **Ember & Origin**, built to establish foundational HTML/CSS layout skills.

## Tech Stack

- HTML5
- CSS3 (no JavaScript — static page)
- Google Fonts: Fraunces (display), Inter (body), IBM Plex Mono (data/labels)

## Features

- Sticky navigation bar with 3 links (Origins, Process, Visit) and a CTA button
- Hero section with headline, subheadline, and call-to-action, alongside a custom SVG "roast curve" graphic (drum temperature vs. time, with a first-crack marker) as the page's signature visual element
- **Origins** section — 3 featured coffee batches with origin, altitude, and tasting notes
- **Process** section — the actual roasting stages (charge → yellowing → first crack → drop), tied to a second roast-curve graphic used as a structural divider
- **Visit** section — address, hours, and a map placeholder
- Footer with placeholder contact and social links
- Fully responsive layout using CSS Grid and Flexbox (breakpoints at 860px and 560px)
- Consistent color system: near-black roast brown, parchment, copper, sage, and gold accents
- Accessible focus states and `prefers-reduced-motion` support
