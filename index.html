Behind the counter

The same curve, logged every time.

We chart drum temperature from charge to drop so every batch can be repeated — or deliberately changed.

~180°C

Charge

Green beans hit the drum. Temperature drops, then climbs.

~160°C

Yellowing

Beans lose grassy notes and start to smell like toast.

196°C

First crack

An audible snap. Development time is measured from here.

204°C

Drop

Beans hit the cooling tray. The curve — and the batch — is logged.
