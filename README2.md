# Web Dev L1 — Task 2: Personal Portfolio

**Track:** Web Development & Designing — Level 1
**Task:** Personal Portfolio
**Author:** Shubham Yadav

## Objective

A personal portfolio website that serves as a digital résumé — showcasing background, skills, and live projects, built with a "build log" visual language (versioned project cards, monospace data labels) that reflects an active, shipping builder rather than a static template.

## Tech Stack

- HTML5
- CSS3 (JavaScript not required — smooth scroll handled natively via CSS)
- Google Fonts: Newsreader (display), Work Sans (body), JetBrains Mono (data/version labels)

## Features

- Hero/profile section: name, role, and photo/avatar placeholder
- About Me: background as a first-year CSE student at KIT Varanasi, Academic Representative, and Techno Club Coordinator
- Skills section split into two honest categories:
  - **Currently hands-on with** — HTML5, CSS3, basic C, Git & GitHub
  - **Builder toolkit** — product thinking, prompt engineering, AI-assisted development, rapid prototyping
- Projects section with 3 versioned project cards (AVEN v2, AVEN v1, AnnadataSetu), each with description, tags, and a GitHub/live-demo link placeholder
- Contact section: name, email, LinkedIn, and GitHub links
- Smooth scroll navigation between all sections
- Consistent branding (color palette, typography) throughout
- Fully responsive — usable on both desktop and mobile (breakpoints at 820px and 540px)
- Accessible focus states and `prefers-reduced-motion` support
